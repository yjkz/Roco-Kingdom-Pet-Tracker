# 精灵计数器桌面悬浮窗 - 技术栈

## 架构总览

| 层面 | 技术 | 说明 |
|------|------|------|
| **语言** | Python 3 | 唯一开发语言 |
| **GUI 框架** | PySide6 | Qt for Python，跨平台桌面 UI |
| **数据模型** | `@dataclass` | Python 标准库，简洁的类型化数据类 |
| **序列化** | JSON | 内置 `json` 模块，可读性好，跨平台通用 |
| **持久化** | 本地文件 (`data/data.json`) | 原子写入（`.tmp` → `os.replace`），插件目录内存储 |
| **主题系统** | 纯 Python dict + CSS | 颜色调色板以 dict 组织，运行时拼接 Qt Stylesheet |
| **窗口控制** | `FramelessWindowHint` + `Tool` | 无边框工具窗口，始终置顶 |
| **圆角** | `QPainterPath` + `setMask` | 系统层裁剪获取真正圆角窗口 |
| **拖拽** | 自定义 `TitleBar` 类 | 重写 `mousePressEvent`/`mouseMoveEvent` |
| **信号通信** | PySide6 `Signal` | 组件间解耦通信（计数变化、精灵增删） |
| **防抖保存** | `QTimer.singleShot` | 500ms 延迟自动保存 |
| **图像处理** | `QPixmap` + `QPainter` | 缩略图缩放、彩色占位符绘制 |
| **图标来源** | 本地 PNG（`570+Icons-CN-v1.0.3`） | 设置齿轮图标 |
| **精灵立绘** | 本地 PNG（`LUOKE DATE/`） | 38 只异色成年体精灵，相对路径引用 |

## 依赖项

```
PySide6>=6.6.0
```

## 项目结构

```
models/     pokemon_data.py    # 数据模型 (dataclass)
core/       persistence.py     # JSON 读写
            app_state.py       # 状态管理 + Signal
            theme.py           # 主题调色板 + 样式生成
views/      overlay_window.py  # 悬浮窗 (TitleBar + ScrollArea)
            pokemon_card.py    # 精灵卡片组件
            settings_dialog.py # 设置面板 (3-tab)
            config_dialog.py   # 精灵编辑对话框
resources/  default_config.json # 38 只预设精灵
```

## 功能清单

| 功能 | 实现方式 |
|------|----------|
| 半透明悬浮窗 | 无边框 + 始终置顶 + `setMask` 圆角 |
| 拖拽移动 | 自定义 TitleBar 接入系统鼠标事件 |
| +/-1 计数 | AppState Signal → PokemonCard 实时更新 |
| 精灵缩略图 | QPixmap 缩放 + 文件缓存 |
| 数据持久化 | JSON 写入本地 `data/` 目录，500ms 防抖 |
| 透明度调节 | 设置面板滑块，stylesheeet 动态调整 |
| 主题切换 | 深色/浅色两套调色板，`_apply_theme()` 统一刷新 |
| 精灵管理 | 添加/编辑/删除，图片浏览，启用/禁用切换 |
| 快速切换显示 | 设置面板 QListWidget 复选框一键开关 |
| 重置所有计数 | 确认对话框后批量归零 |
| 窗口宽度调整 | 设置面板滑块，280-500px 范围 |
